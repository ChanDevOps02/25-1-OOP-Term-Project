=== File Manager Pro 도움말 ===

1. 확장자별 정리
   • "확장자별 정리" 버튼 클릭
   • 폴더 선택 후 "실행" → 해당 폴더의 파일이 확장자별 폴더로 이동됩니다.
   • "정리 취소"로 바로 복원 가능.

2. 미사용 파일 삭제
   • "미사용 파일 삭제" 버튼 클릭
   • 폴더 선택, 기간(일) 입력 후 "검색" → 미사용 파일 목록 표시
   • "전체 삭제" 버튼으로 한 번에 삭제

3. 디스크 용량 확인
   • 프로그램 시작 시 상단에 "디스크 사용량: 사용 / 전체" 표시

4. 고급 검색
   • "고급 검색" 클릭
   • 폴더 선택, 파일 이름 키워드, 크기(KB), 수정일 범위 입력 후 "검색 실행"
   • 조건에 맞는 파일 목록 표시

5. 사용 이력 시각화
   • "사용 이력 시각화" 클릭
   • 폴더 선택, 기간(일) 입력 후 "시각화 실행"
   • 지난 N일간 일별 수정 파일 개수를 막대차트로 표시

6. 암호화/복호화
   • "암호화/복호화" 클릭
   • 파일 선택, 비밀번호 입력(확인), "🔒 암호화" 또는 "🔓 복호화" 클릭
   • 진행률 바와 상태 로그 제공

7. 추가 설명
   • 모든 기능은 우측 메인 화면에서 실행 결과를 확인하세요.
